⭐ Recomendado — limpo + discoverability no sheet

C — Primários + overflow "···" ⭐

3 ações principais visíveis (Gabarito, Stats, Comunidade) + botão "···" que abre bottom sheet com Nota, Flag, IA, Bookmark. Limpo, touch-friendly, e as ações secundárias ganham label + descrição no sheet.

Por que C é melhor?

Vantagens

  • Footer compacto — 1 linha, mesma altura do desktop
  • Touch targets de 36px+ em vez de 15px
  • Bottom sheet dá espaço pra label + descrição (discoverability)
  • Funciona igual no tablet e celular
  • Padrão conhecido (Instagram, Twitter, YouTube)

Trade-offs

  • 1 toque extra pra acessar ações secundárias
  • Mais um componente (bottom sheet) pra manter
