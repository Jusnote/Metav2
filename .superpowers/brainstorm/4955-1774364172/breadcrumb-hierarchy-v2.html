A — Indent lines + Level badges

Linhas verticais de conexão em verde sutil marcam a profundidade. Badges coloridos (Parte, Tít, Cap) identificam o nível instantaneamente. Artigos com dot indicator, ativo com barra lateral verde.

B — Nested cards com contadores

Cada nível é um card aninhado. Contadores ("3 títulos", "6 caps") dão contexto de volume. Cards dentro de cards demonstram visualmente a profundidade. Ativo com background verde.

C — Connected tree (graph visual)

Circles + linhas conectoras como um grafo/árvore real. Tipo hierarchy de nível em VS Code ou org chart. Cada nó é um ponto conectado ao pai. Ativo pulsa com ring verde.
