Features de UX que elevam a experiência

Além do layout, funcionalidades que diferenciam de qualquer app jurídico. Selecione as que fazem sentido para o v1.

1

Toggle "Lei Seca"

Um switch no topo que esconde todas as anotações legislativas "(Incluído pela Lei nº...)", mostrando só o texto puro da lei. Estudantes de concurso precisam ler a lei "seca" — sem anotações. Toggle instantâneo, sem reload.

COM anotações
Art. 121. Matar alguem:
(Vide Lei nº 8.072, de 25.7.90)
LEI SECA (limpo)
Art. 121. Matar alguem:
2

Referência Cruzada Inline

Quando o texto menciona "nos termos do art. 121", o número vira um link clicável. Hover mostra um tooltip com o texto do artigo referenciado — sem sair da página. Os dados de links[] da API já contêm essas referências.

...responde o agente pelo crime de homicídio, nos termos do art. 121 deste Código.
Art. 121 — Homicídio simples
Matar alguem: Pena - reclusão, de seis a vinte anos.
3

Busca com Minimap (estilo VS Code)

Ctrl+F abre barra de busca que usa a API busca() com websearch_to_tsquery. Resultados aparecem como marcadores na barra de scroll lateral — clique para pular direto. Suporta "prisão preventiva" -fiança.

🔍 homicídio qualificado 22 resultados
Art. 121, § 2º — Se o homicídio é cometido...
Art. 121, § 2º-B — A pena do homicídio contra menor...
4

Navegação por Teclado

Atalhos para quem estuda de verdade:

J K próximo/anterior artigo
Ctrl+\ toggle modo imersivo
Ctrl+G ir para artigo (nº)
L toggle lei seca
5

Progresso de Leitura

Barra fina no topo mostra % lida da lei. Persiste no localStorage. Quando volta para a lei, scrolla automaticamente para onde parou. A sidebar marca artigos já visitados com um dot sutil.

34% lido · Art. 121 / 361 artigos ↩ Continuar de onde parou
6

Toggle Revogados

Artigos revogados ficam colapsados por default com um indicador visual (Revogado). Clique para expandir e ver o texto original riscado. A API já filtra com incluirRevogados.

▶ § 3º Se o homicídio é culposo: Revogado

Recomendação para v1

Essenciais: 1 (Lei Seca toggle), 4 (teclado), 6 (revogados) — baixo esforço, alto impacto
Diferenciais: 3 (busca minimap), 5 (progresso) — médio esforço, experiência premium
v2: 2 (ref cruzada) — precisa de dados de links entre leis, mais complexo
