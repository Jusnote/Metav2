Anotações — Ainda São Inline (Mesmo Problema)

Anotações NÃO estão em tags separadas no JusBrasil
// HTML real do Art. 3º no JusBrasil:
<a href="...">Art. 3º</a>
São absolutamente incapazes... 16 anos.
(Redação dada pela Lei nº 13.146, de 2015) (Vigência)

// As anotações são texto puro no final do <p>
// NÃO estão em <a>, <span> ou qualquer tag especial
// 0 anotações em links, TODAS como texto puro
Implicação
JusBrasil tem a mesma limitação de anotações que o Planalto — são texto inline. O regex de separação ainda é necessário. MAS o JusBrasil tem uma vantagem enorme: cada dispositivo já está isolado em seu próprio <p>.

Veredicto: JusBrasil vs Planalto

Critério Planalto JusBrasil
Cada dispositivo é entidade própria Não (HTML varia) Sim (cada <p>)
Anotações separadas do texto Não (inline) Não (inline)
Marca dispositivos revogados Parcial Sim (CSS class)
Link por dispositivo (âncoras) Varia por lei Sim (toda <a>)
HTML consistente entre leis Não Sim (padrão único)
Todas as leis na mesma página Sim Sim
Acesso automatizado HTTP direto 403 HTTP / OK via browser
Fonte canônica (oficial) Sim (gov) Não (privado)
Risco legal de scraping Zero (gov) Baixo (lei é pública)

Proposta Revisada: JusBrasil (primário) + Planalto (validação)

1. Fonte Primária — JusBrasil via Browser Automation
   • Cada dispositivo já isolado em <p> — parsing trivial
   • HTML consistente entre leis — um parser pra todas
   • Versões revogadas marcadas com CSS class
   • Playwright/Puppeteer para contornar o 403

2. Separação de Anotações — Regex Aprimorado + LLM
   • As anotações são inline tanto no JusBrasil quanto no Planalto
   • Regex expandido para cobrir TODOS os padrões (Regulamento, Produção de efeito, etc.)
   • Para os casos ambíguos: LLM (Claude API) como fallback de classificação
   • Armazenar: texto_limpo + anotacoes[] + texto_original

3. Validação Cruzada — Planalto como Fonte de Verdade
   • Fetch do Planalto HTML para comparar contagem de dispositivos
   • Amostragem aleatória de texto para detectar divergências
   • Planalto É a fonte oficial — se JusBrasil divergir, flag automático

4. Relatório de Qualidade
   • Score de confiança por lei e por dispositivo
   • Regex de segurança no texto_limpo (anotações residuais)
   • Revisão manual apenas dos itens flagados
