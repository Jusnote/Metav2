Componentes do Sistema

Componente Tecnologia Responsabilidade
Scripts CLI Node.js puro extract.js (GraphQL → raw.json), process.js (raw → processed), upload.js (→ PostgreSQL + Typesense)
API GraphQL Fastify + Mercurius Servir leis, dispositivos, hierarquia. Proxy busca para Typesense.
PostgreSQL PostgreSQL 16 Source of truth. Tabelas: leis, dispositivos. Índices GIN para tsvector.
Typesense Typesense (existente) Busca instant com typo tolerance, facets, highlight. Collection: dispositivos.
Frontend React + Next.js (Metav2) Renderização pura de dispositivos. getSelection() para grifos. TipTap leve para notas.
Supabase Supabase (existente) Auth, grifos do usuário, anotações, flashcards, cadernos. Real-time sync.
